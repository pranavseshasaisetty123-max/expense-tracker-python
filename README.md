# Expense Tracker (Python)

A console-based Expense Tracker built using Python.

## Features

* Add Expense
* View Expenses
* Delete Expense
* Category-wise Expense Tracking
* Search Expenses by Category
* Total Spending Calculation
* Category Summary
* Save Data to File
* Auto Load Data on Startup

## Concepts Used

* Variables
* Lists
* Dictionaries
* Functions
* Loops
* Conditional Statements
* File Handling
* Exception Handling

## How to Run

```bash
python expense_tracker.py
```

## Project Goal

This project helps users record and manage daily expenses while practicing core Python programming concepts.
